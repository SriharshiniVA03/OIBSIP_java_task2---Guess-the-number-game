# Guess Number Game (Java)

## Internship Task
OIBSIP - Task 2

## Objective
The objective of this project is to build a simple number guessing game in Java where the computer randomly generates a number and the user tries to guess it within a limited number of attempts.

## Description
This program generates a random number between 1 and 100.  
The user must guess the number within 5 attempts.  

The program provides hints:
- "Too High" if the guess is greater than the number
- "Too Low" if the guess is smaller than the number

The game runs for 3 rounds and calculates the total score based on the number of attempts used.

## Features
- Random number generation
- Limited number of attempts
- Multiple rounds
- Score calculation
- User feedback (Too High / Too Low)

## Tools and Technologies Used
- Java
- Java Random Class
- Java Scanner Class
- Command Line Interface

## Steps Performed
1. Imported required Java packages (Scanner and Random).
2. Generated a random number between 1 and 100.
3. Allowed the user 5 attempts to guess the number.
4. Provided hints for each incorrect guess.
5. Calculated score based on remaining attempts.
6. Repeated the game for 3 rounds.
7. Displayed final score at the end of the game.

## Outcome
The program successfully allows the user to play a number guessing game with multiple rounds and scoring, improving understanding of loops, conditions, and random number generation in Java.
